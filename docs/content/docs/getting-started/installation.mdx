#

## Download

Looma is distributed as a desktop application. You can download Looma from:

- 🌐 **Official website**
https://loomachat.com

- 🐙 **GitHub releases**
https://github.com/victor-YT/looma/releases

---

## Install

After downloading the installer:

1. Open the downloaded file.
2. Follow the installation instructions for your operating system.
3. Launch Looma.
