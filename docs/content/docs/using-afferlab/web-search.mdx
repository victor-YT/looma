title: Web Search
description: Web search allows AfferLab conversations to access real-time information using model-native capabilities.
---

#

## How It Works

Web search can be enabled for a conversation using the **Web Search icon** below the input area.

When enabled, AfferLab attempts to use the selected model’s **native web search capability** during the conversation.

---

## Model Native Search

Some models provide **built-in web search support**.

When a compatible model is selected:

- AfferLab enables the model’s **native search mode**
- The model can retrieve up-to-date information directly during generation
- No additional tools or configuration are required

---

## When Search Is Not Available

If the selected model **does not support native web search**, AfferLab will:

- Display an **error message** indicating that search is not available
- Continue the conversation **normally without search**

This means:

- Chat functionality is not affected
- Only web search is disabled

---

## Notes

- Web search availability depends entirely on the **selected model**
- Different models may provide different levels of search quality and coverage
