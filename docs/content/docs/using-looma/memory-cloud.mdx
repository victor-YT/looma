#

## Overview

Memory Cloud is a user-managed asset database that can be enabled by strategies.
Users can store and organize assets in the cloud, while strategies define how those assets are retrieved and used during conversations.

## When It Appears

Memory Cloud is not globally available.

It is only enabled when the active strategy declares the Memory Cloud feature.

## How Strategies Use It

Strategies control how Memory Cloud assets participate in a conversation.

A strategy may choose to:

- retrieve assets as additional context
- ignore stored assets entirely
- selectively load assets based on the conversation state

This keeps the platform flexible while allowing different strategies to implement different memory behaviors.
