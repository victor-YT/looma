#

## Current Sections

The settings currently includes the following sections:

- Model
- Model Settings
- Strategy
- Web Search
- General
- Data & Privacy

## Model

- configure provider
- enable or disable models
- test provider/model connectivity
- add custom provider models

## Model Settings

Model settings allow you to adjust default generation behavior such as temperature, top p and output token limits.

## Strategy

Strategy settings allow you to write, manage available strategies and configure parameter.

## Web Search

Web search settings control whether web search is enabled and how many results are returned during searches.

## General

General settings include application preferences such as theme mode, launch behavior, and auto-scroll behavior.

## Data & Privacy

Data and privacy settings control how Looma stores local data and manages privacy-related preferences.
